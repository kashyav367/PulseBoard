# PulseBoard 🚀

PulseBoard is a real-time live polling platform where users can create polls, vote instantly, and view live analytics.

---

# 🌐 Live Demo

## Frontend
https://pulse-board-ebon.vercel.app

## Backend
https://pulseboard-o4dg.onrender.com

---

# ✨ Features

- Google Authentication
- JWT Authentication
- Create Live Polls
- Real-time Voting
- Live Analytics Dashboard
- Responsive UI
- Protected Routes
- MongoDB Database

---

# 🛠️ Tech Stack

## Frontend
- React.js
- Vite
- Tailwind CSS
- Axios
- React Router DOM

## Backend
- Node.js
- Express.js
- MongoDB
- Passport.js
- JWT

# 🚀 Run Locally

## Clone Repository

```bash
git clone https://github.com/kashyav367/PulseBoard.git
```

---

## 💻 Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## ⚙️ Backend Setup

```bash
cd backend
npm install
npm run server
```
